El Terror Psicológico

¿Qué harías si el silencio fuera tu única defensa?

ProjectDusk no es solo un juego de terror de supervivencia; es una experiencia de inmersión total en la audición. Eres un investigador en un complejo subterráneo después de un misterioso incidente sonoro. Las criaturas que te acechan no ven, solo escuchan. Cada paso, cada respiración, cada fallo eléctrico puede ser el último.

  • Inmersión Sonora 3D.
  • IA de escucha dinámica.
  • Múltiples finales narrativos.

Requisitos del Sistema

Mínimos

  • **SO:** Windows 10 (64-bit)
  • **CPU:** Intel Core i5 / AMD Ryzen 5
  • **RAM:** 8 GB
  • **GPU:** NVIDIA GeForce GTX 1060 / AMD Radeon RX 580
  • **Almacenamiento:** 50 GB de espacio disponible
  • **Sonido:** Tarjeta de sonido compatible con Audio 3D (Recomendado)

Recomendados

  • **SO:** Windows 11 (64-bit)
  • **CPU:** Intel Core i7 / AMD Ryzen 7
  • **RAM:** 16 GB
  • **GPU:** NVIDIA GeForce RTX 3070 / AMD Radeon RX 6700 XT
  • **Almacenamiento:** SSD, 50 GB de espacio disponible
  • **Sonido:** Auriculares de alta calidad para Audio Espacial

*Los requisitos están sujetos a cambios. Se recomienda una conexión a Internet para actualizaciones.
